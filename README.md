# denite-markdown

denite-markdown is [denite.nvim](https://github.com/Shougo/denite.nvim) source for markdown

## Install

For [vim-plug](https://github.com/junegunn/vim-plug)

```vim
Plug 'N0nki/denite-markdown'
```

For [dein.vim](https://github.com/Shougo/dein.vim)

```vim
call dein#add('N0nki/denite-markdown')
```

## Usage

List up all headers

```vim
:Denite markdown
```
